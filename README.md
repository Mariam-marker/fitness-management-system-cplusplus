# fitness-management-system-cplusplus
C++ Fitness Management System for Structured Programming Course

A comprehensive fitness management system developed as a **group project** for a college **Structured Programming (C++)** course.
This project simulates a modern fitness center with features for clients, trainers, and an online coaching module.

---

## 🧠 Features

### ✅ Trainer Dashboard
- Register and login.
- Create workout plans with multiple exercises.
- Assign workouts to clients.
- View client progress and feedback.

### ✅ Client Dashboard
- Register and login.
- View assigned workouts.
- Log completed workouts and body measurements.
- Provide feedback to trainers.

### 🏋️ Online Coaching Module
- Calculates BMR (Basal Metabolic Rate) and TDEE (Total Daily Energy Expenditure).
- Suggests personalized diet and macronutrient breakdowns.
- Offers diet guidance for goals: maintain, bulk, or cut.
- Includes tips for hydration, nutrition, and training.
